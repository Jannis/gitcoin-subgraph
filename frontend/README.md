# PoC frontend for Gitcoin Subgraph

Simple UI to demonstrate Gitcoin subgraph indexing capabilities such as bounties, bounty funders/hunters, and Kudos NFTs

### Deployment

* To deploy this app:
  ```sh
  yarn && yarn build
  yarn global add serve
  yarn serve -s build
  ```

#### Setup for development

* To start this app:
   ```sh
   yarn install
   yarn start
   ```
